وَسَد - النسخة المدمجة

تم دمج واجهات الطفل داخل المشروع الأساسي مع الإبقاء على واجهات ولي الأمر.

التشغيل:
1) افتحي المجلد في VS Code.
2) افتحي Terminal داخل المجلد.
3) شغلي: npm install
4) ثم: npx expo start -c
5) اختاري Android أو امسحي QR من Expo Go.

المسارات:
- الدخول كولي أمر: /parent
- الدخول كطفل: /child ثم ينتقل إلى /home
- صفحات الطفل: home, quests-child, wallet-child, goals-child, pet-child, stories-child, pet-shop-screen

ملاحظة: لا تنقلي مجلد app أو src أو assts خارج مكانه لأن مسارات الصور تعتمد عليها.
