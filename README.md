# وَسَد — Clean Starter

مشروع Expo Router نظيف كبداية جديدة.

## التشغيل

افتحي المجلد داخل VS Code ثم شغلي:

```bash
npm install
npx expo start
```

بعدها:
- اضغطي `a` لتشغيل محاكي أندرويد.
- أو امسحي QR من Expo Go إذا كانت النسخ متوافقة.

## الموجود حاليًا

- شاشة اختيار الدور.
- صفحة مؤقتة لولي الأمر.
- صفحة مؤقتة للطفل.
- ألوان ومسافات موحدة.
- Safe Area مضبوط.
- لا يوجد Context أو Provider معقد.
